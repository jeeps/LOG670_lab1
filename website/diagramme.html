Explications du modèle du domaine

  • Centrale
    La centrale est le gestionnaire principale. C'est cette classe qui permet de communiquer avec chacun des camions individuellement ou avec les convois. Elle à également le rôle de suiveiller le réseau et gérer les routes.

    Les associations :

    1. → Camion
    2. → Convoi
    3. → Balise
    4. → Route
